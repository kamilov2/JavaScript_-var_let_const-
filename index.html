JavaScript let, var, const bo'yicha qo'llanma

1. Var

var o'zgaruvchi e'lon qilish uchun ishlatiladi. var funksion doirada (function scope) ishlaydi va bir xil doirada bir necha marta e'lon qilinishi mumkin.

        
var x = 10;
console.log(x); // 10

if (true) {
    var x = 20;
    console.log(x); // 20
}

console.log(x); // 20

2. Let

let blok doirasida (block scope) ishlaydi va bir xil doirada faqat bir marta e'lon qilinishi mumkin.

        
let y = 10;
console.log(y); // 10

if (true) {
    let y = 20;
    console.log(y); // 20
}

console.log(y); // 10

3. Const

const o'zgarmas o'zgaruvchi e'lon qilish uchun ishlatiladi. Qiymati e'lon qilinganidan keyin o'zgartirilishi mumkin emas. const ham blok doirasida (block scope) ishlaydi.

        
const z = 10;
console.log(z); // 10

if (true) {
    const z = 20;
    console.log(z); // 20
}

console.log(z); // 10

// Quyidagi kod xato beradi:
// z = 30;

4. Let, var va const o'rtasidagi farqlar

Quyidagi jadvalda let, var va const o'rtasidagi asosiy farqlar ko'rsatilgan:

Farqlar var let const
Scope (Doirasi) Funksion Blok Blok
Re-e'lon qilish (Qayta e'lon qilish) Ha Yo'q Yo'q
Qiymatini o'zgartirish Ha Ha Yo'q
Oldin ishlatish Hoisting (undefined bilan) Hoisting (Error bilan) Hoisting (Error bilan)

Misollar:

Var:

        
console.log(a); // undefined
var a = 10;
console.log(a); // 10

Let:

        
try {
    console.log(b); // ReferenceError
} catch (e) {
    console.log(e.message); // b is not defined
}
let b = 20;
console.log(b); // 20

Const:

        
try {
    console.log(c); // ReferenceError
} catch (e) {
    console.log(e.message); // c is not defined
}
const c = 30;
console.log(c); // 30

try {
    c = 40; // TypeError
} catch (e) {
    console.log(e.message); // Assignment to constant variable.
}
